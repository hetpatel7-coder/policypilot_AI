🚀 PolicyPilot

PolicyPilot is an AI-powered system that helps citizens easily find and apply for government welfare schemes. Many people miss benefits because they don't understand eligibility rules or application processes.

The system allows users to describe their situation in simple language or voice, and the AI finds all schemes they qualify for.

🎯 Key Features
Natural Language Input – Users describe their situation (e.g., farmer, income, state).
Scheme Matching – AI identifies government schemes the user is eligible for.
RAG-based Document Search – Uses official scheme PDFs for accurate results.
Application Checklist – Shows required documents and steps to apply.
Auto-filled Forms – Generates draft application forms using uploaded documents.
Conflict Detection – Detects contradictory rules between central and state schemes.
Multilingual Support – Provides results in Hindi or regional languages.
🧠 Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Python / Node.js
AI: LLM + RAG (LangChain / LlamaIndex)
Vector Database: FAISS / Pinecone
Document Processing: OCR + PDF parsing
⚙️ How It Works
User enters their profile information.
AI extracts key details.
RAG searches government scheme PDFs.
System finds eligible schemes.
AI generates checklist and application guidance.
